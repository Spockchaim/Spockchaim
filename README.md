### 👤 Sobre Mim

Sou estudante de **Desenvolvimento de Software Multiplataforma** na **FATEC São José dos Campos** (2025 - 2028). Busco constantemente expandir meus conhecimentos no ecossistema de desenvolvimento de software e análise de dados.

Minha trajetória une a experiência analítica e de negócios adquirida nas áreas comercial e administrativa com o entusiasmo pela inovação tecnológica. Acredito que a tecnologia deve ser usada para resolver problemas reais e otimizar processos de forma inteligente e eficiente.

* 🚀 **Foco Atual:** Desenvolvimento Back-End (Python, Java, Kotlin) e Ciência de Dados.
* 🎓 **Educação:** FATEC SJC & cursos complementares pela Oracle/SENAC, SENAI e FGV.
* 🌐 **Portfólio Web:** Acesse o meu portfólio profissional para ver meus projetos em funcionamento (em breve online).
